О программе

Данное приложение предназначено для управления систем гостинцы "Афродита". В нем реализован удобный интерфейс для работы с основными сущностями отеля:

  • Бронирование — управление бронями: их создание, редактирование и удаление.
  • Номера — Просмотр добавленных в систему номеров отеля, их текущего статуса и состояния
  • Постояльцы — Просмотр занесённых в систему постояльцев отеля. Можно будет удалять, редактировать и добавлять постояльцев
  • Организации — инструменты для отслеживания текущих скидок для организаций и их дат начала/окончания.
  • Жалобы — управление жалобами, дата занесения их в систему.
  • Дополнительные услуги — управление дополнительными услугами и их ценами.

Основные функции приложения

  • Поиск для быстрого доступа к нужным данным.
  • Редактирование и удаление существующих данных с подтверждением действий.
  • Роли и права доступа, обеспечивающие безопасность и разграничение функционала.
  • Навигация по разделам через нижнее меню, доступное на всех страницах.

Технологии

Приложение построено на Spring Boot с использованием Thymeleaf для серверного рендеринга страниц и Bootstrap 5 для адаптивного и современного интерфейса.

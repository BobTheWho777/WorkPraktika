Добро пожаловать в систему управления гостиницей

Это веб-приложение позволяет администратору управлять гостиничными бронированиями, номерами, постояльцами, организациями, жалобами и дополнительными услугами. Пользователи с ролью USER могут только просматривать данные, не редактируя их.

1. Авторизация
  • Перейдите на страницу входа /login.
  • Введите логин и пароль, выданные администратором.
  • После входа вы будете перенаправлены на список бронирований.
2. Навигация по разделам

Внизу каждой страницы доступно меню перехода к разделам:

  • Бронирования — /reservations
  • Номера — /rooms
  • Постояльцы — /guests
  • Организации — /organizations
  • Жалобы — /complaints
  • Дополнительные услуги — /additional-services
3. Возможности в зависимости от роли
  • ADMIN — полный доступ (создание, редактирование, удаление, просмотр).
  • USER — доступ только к просмотру данных. Кнопки «Добавить», «Редактировать», «Удалить» не отображаются.
4. Работа с разделами
4.1. Добавление
  • Доступно только администраторам.
  • Кнопка «Добавить» находится над таблицей.
  • После заполнения формы нажмите «Создать».
4.2. Редактирование
  • Для изменения записи нажмите кнопку «Редактировать» в строке нужной записи.
  • Внесите изменения в форму и нажмите «Сохранить изменения».
4.3. Удаление
  • Нажмите кнопку «Удалить» в строке нужной записи.
  • Подтвердите удаление в всплывающем окне.
4.4. Поиск
  • В каждом разделе вверху есть поле поиска.
  • Введите нужное имя, номер или фразу и нажмите «Поиск».
5. Выход из системы
  • Внизу каждой страницы находится кнопка «Выход».
  • Нажмите её, чтобы завершить текущую сессию.
6. Общие советы
  • Все обязательные поля форм помечены и требуют заполнения.
  • Поля «Жалоба» и «Дополнительная услуга» в бронировании можно оставить пустыми.
  • Следите за корректностью введённых дат и числовых значений (например, цена, количество человек, долг).
